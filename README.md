## 🚀 Featured Projects

<details open>
<summary><b>🎯 SHL Assessment Recommendation System</b></summary>
<br/>

A retrieval-augmented (RAG) recommendation engine that maps natural language job descriptions to the most relevant SHL assessments — built to help recruiters replace slow keyword search with semantic search.

| | |
|---|---|
| **Stack** | Python, FastAPI, SentenceTransformers, FAISS, Pandas, Jinja2 |
| **Scale** | 389+ SHL Individual Test Solutions indexed and searchable |
| **Performance** | Sub-second nearest-neighbor retrieval via FAISS vector search |
| **Security** | REST API compliant with SHL specification standards |
| **Impact** | Replaces manual keyword filtering with balanced, semantic recommendations across technical and behavioral assessment types |
| **Repository** | [SHL-Assessment-Recommender](https://github.com/therealsheero/SHL-Assessment-Recommender) · [Live Demo](https://shl-assessment-recommender-yhxi.onrender.com/) |

Built a full retrieval pipeline — sentence embeddings (`all-MiniLM-L6-v2`) → FAISS vector similarity search → ranking → test-type balancing — backed by a self-scraped, structured catalog of SHL assessments. Evaluated against labeled queries using Mean Recall@K, with a live FastAPI backend and a lightweight Jinja2 frontend deployed end-to-end on Render.
